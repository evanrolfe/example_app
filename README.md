# Example App

**Installation**

```bash
$ git clone git@github.com:evanrolfe/example_app.git && cd example_app
$ docker-compose up
$ docker-compose exec backend rake db:setup
```

Access the SPA from http://localhost and the MPA from http://localhost:3001.
